# Public-Transportation-Database
A MySQL database project simulating a real-world public transportation system with advanced SQL features.

## Overview
This project models the core components of a city transportation system, including passengers, vehicles, routes, trips, and maintenance processes.
It demonstrates database design, data integrity, and advanced SQL usage in a realistic scenario.

## Schema
The database includes the following entities:
- Passenger
- Card
- Vehicle
- Driver
- Route
- Stop
- Route_Stop
- Trip
- Tap
- Maintenance
- Maintenance_Type
Relationships are implemented using primary and foreign keys to ensure consistency.

## Features
- Normalized relational schema  
- Sample data for all tables  
- Constraint-based validation  
- Advanced SQL usage:
  - JOINs
  - Subqueries
  - Aggregations
  - GROUP BY / HAVING
  - Views
  - Stored Procedures
  - Triggers  

## Usage
Run the SQL script in MySQL:
SOURCE Public Transportation Database.sql;
This will automatically create and populate the database.
